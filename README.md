# 🌌 Astroway Pro – Astrologer App

Astroway Pro is a professional Flutter application designed for astrologers to connect with clients through live streaming, chat, and personalized astrology services.

---

## ✨ Features

- 🔮 **Astrology Services** – Provide consultations, daily horoscopes, and kundli generation.
- 📞 **Live Audio/Video Calls/Chats** – Secure and high-quality live streaming powered by Zego Cloud , Agora , hms.
- 💬 **Chat System** – Real-time messaging with users using Firebase.
- 💳 **In-App Purchases** – Easy and secure payment integration for sessions/packages.
- 🌍 **Multi-language Support** – Choose preferred language for wider audience reach.
- 📊 **Dashboard for Astrologers** – Manage profile, availability, and consultation history.
-   **Coin/currency** - wallet switch between coin or currency system

---

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (**3.35.3** or later recommended)
- **Android Studio / Xcode**
- **ZegoCloud** account (for live streaming features)
- **AGORA** account (for live streaming features)
- **HMS or 100ms** account (for live streaming features)
- **Firebase** (for push notifications & authentication)
